Beyond OSU II ePortfolio

This page is my ePortfolio for the Beyond OSU II Core Requirement at Oregon State University. The ePortfolio is the Essential Assignment for this requirement: it compiles writing samples, team work, and career documents. This version uses a public professional website so reviewers and employers can open materials without sign-in or permission requests (unlike some shared-drive links).

How this ePortfolio is organized

Materials are grouped by type and assignment role. Each item below has a clear title, a short description of what the file is and why it is included, and a link. File names on the server are kept simple and descriptive (for example, resume.pdf, coverletter.pdf, designproposal.pdf) so the purpose is obvious—avoiding unclear names like “498_DP-final-final.”

Additional artifacts (course plan — end of term)

The full Beyond OSU II ePortfolio, when complete, will also include team assignments and reflective work. Those items are not required for the ePortfolio Start submission but are listed here to show how this site will stay organized as they are added:

  • Individual self-learning assignment — to be added with a descriptive title and link when finalized.
  • Final team report — to be added when available.
  • Team charter — to be added when available.
  • Reflection essay — map of experiences to NACE Career Readiness Competencies and future goals; to be added when finalized.

Each future artifact will follow the same pattern: clear name, short context, and a public link or embedded PDF on this page.
